# Screenshots

Drop the walkthrough images here. The main [README](../../README.md#walkthrough)
references these exact file names — capture each screen and save it with the
matching name (PNG, ~1400px wide is plenty).

## SaaS Admin Console (provider)

| File | What to capture |
|------|-----------------|
| `console-templates.png` | **Templates** tab — the agent template catalog (publish/unpublish). |
| `console-customers.png` | **Customers** tab — onboard a customer (org_id, tier/quota, branding). |
| `console-instances.png` | **Instances** tab — assign a template, upload knowledge, toggle agentic retrieval. |
| `console-metering.png`  | **Metering** tab — per-customer tokens/calls charts and usage by instance. |
| `console-costs.png`     | **Costs** tab — platform + AI usage total, per-customer split, projection (USD/EUR). |

## Customer chat (end user)

| File | What to capture |
|------|-----------------|
| `customer-welcome.png`  | Welcome screen with the brandable header and suggested-question chips. |
| `customer-answer.png`   | A grounded answer (RAG) rendered as markdown, citing the customer's knowledge. |

> Tip: in the Admin Console the header shows a **DEMO MODE** banner and a
> **Demo admin login** button; you can crop it out or leave it in to make clear
> it's the dev/demo build.
